{% set about %} Welcome to Zwapp! While not intended for "real world" use, Zwapp is meant to be a conceptual app that showcases my abilities as a UI/UX developer. It also serves as my first project using PHP and MongoDB. This app illustrates possible use cases for easily finding a particular comicbook issue, for example to entered into a comicbook collection. Zwapp is also intended as a "mobile-first" application in which the workflows can be intuitively followed using a single finger on a touch screen, while providing good responsiveness for use with a desktop browser.

Because the focus of this app is on user interaction and app front-end design, many of the theoretical workflows will only go so far as to demonstrate the UI. With that in mind, please enjoy!
