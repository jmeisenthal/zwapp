{% set about %} Welcome to Zwapp, a concept-app (prototype) that showcases my abilities as a UI/UX developer!

Zwapp also serves as an example of an app using PHP and MongoDB. This app illustrates possible use cases for easily finding a particular comic book issue, for example to be entered into a comic book collection.

Zwapp also demonstrates a "mobile-first" application in which the workflows can be intuitively followed using a single finger on a touch screen, while providing good responsiveness for use with a desktop browser.

The focus of this app is to demonstrate user interaction and app front-end design, so some of the theoretical workflows only go so far, enough to demonstrate the UI.

With that in mind, please enjoy!

PHP The server side has been implemented in PHP, both as a provider of static html using the Twig templating library, and as microservices used to asynchronously provide queried content. Unit tests were run using PHPUnit.
jQuery Used for much of the Javascript run on the client.
MongoDB Used for permenant storage of data queried from the ComicVine online database.
Node.js Used As a platform to manage Javascript libraries and scripts with npm and build/deploy with WebPack.
